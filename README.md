# Konfigurasi Aplikasi

OpenSID menyediakan berbagai setting konfigurasi untuk menyesuaikan aplikasi dengan kebutuhan desa. Misalnya, ada setting untuk menggantikan sebutan _desa_ dengan kata lain, misalnya _kampung_. Panduan mengubah konfigurasi aplikasi ada di [Panduan Konfigurasi Aplikasi](Panduan-Konfigurasi-Aplikasi).

## Identitas Desa
Di form isian ini, masukkan data dasar desa seperti nama desa, alamat kantor desa, logo desa dsbnya. Petunjuk fitur ini ada di [Petunjuk Pengisian Identitas Desa](Identitas-Desa).

## Pemerintah Desa
Fitur ini dipakai untuk mengelola daftar perangkat desa, seperti kepala desa, sekretaris desa dan sebagainya. Perangkat desa yang terdaftar di laman ini akan tampil sebagai anggota pemerintah desa yang dapat menandatangani surat di modul _Cetak Surat_.

Petunjuk fitur ini ada di [Pengisian Pemerintah Desa](Pemerintah-Desa).

## SID
Laman ini berisi informasi mengenai SID, termasuk tautan sumber informasi seputar OpenSID.

## Cetak Surat

Salah satu fungsi SID yang sangat membantu kantor desa adalah kemampuannya untuk mencetak berbagai surat administrasi kependudukan yang biasa disiapkan oleh kantor desa. Data untuk setiap surat diambil secara langsung dari database penduduk.

Modul _Cetak Surat_ menyediakan fitur untuk mencetak surat, untuk mengelola arsip layanan, untuk mengelola jenis surat yang disediakan dan untuk mengubah format surat sesuai kebutuhan desa.

Panduan modul _Cetak Surat_ ada di [[Cetak Surat]].

## Surat Masuk

Tab _Surat Masuk_ digunakan untuk mengelola surat masuk yang diterima oleh kantor desa. Petunjuk pengelolaan surat masuk menggunakan OpenSID ada di [[Surat Masuk]].

## SK KADES

Tab _SK KADES_ digunakan untuk mengelola arsip SK Kepala Desa. Petunjuk pengelolaan arsip SK Kepala Desa ada di [[SK KADES]].

## PERDES

Tab _PERDES_ digunakan untuk mengelola arsip Peraturan Desa. Petunjuk pengelolaan arsip Peraturan Desa ada di [[PERDES]].

## Inventaris Desa

Tab _Inventaris Desa_ digunakan untuk mengelola inventaris barang dan kekayaan desa, termasuk mengeluarkan _Buku Inventaris dan Kekayaan Desa_ sesuai dengan Permendagri No. 47 Tahun 2016. Petunjuk pengelolaan Inventaris Desa ada di [[Inventaris Desa]].

# <img src="https://cloud.githubusercontent.com/assets/249563/20508969/2d640c4a-b0ca-11e6-9e92-5fa12e9c2bf9.png" width="50px"> Analisis
Menu Analisis tempat mengelola data hasil survei sesaat, seperti Analisis Kemiskinan Partisipatif, atau tempat menyerap data yang perlu dikelola berkesinambungan, seperti Profil Desa dan Kelurahan (Prodeskel).

Data serapan seperti Prodeskel dikelola sebagai analisis bawaan sistem dan sama untuk semua pengguna. Data survei sekali pakai berbeda untuk setiap desa.

Petunjuk pengelolaan Analisis ada di [Pengelolaan Analisis](Analisis).

Modul Web bisa diakses pada http://localhost/OpenSID/index.php/first. Walaupun bisa diakses pada SID _offline_ (yaitu instalasi yang tidak bisa diakses dari internet), umumnya modul Web ditujukan pada instalasi SID _online_, yang bisa diakses melalui internet. Antara lain, isi modul Web adalah untuk mendukung transparansi kegiatan pemerintah desa, menggalang partisipasi warga, promosi produk dan kelebihan desa, dsbnya.
